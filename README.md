# Auralyn – Mood-Aware Music Recommendation System
An AI-driven music recommendation system that adapts playlists based on emotional and contextual cues.
## Features
- Mood-aware music recommendations
- Audio feature analysis using AI
- Personalized playlists
- Interactive frontend interface
- REST API–based backend
## Tech Stack
**AI / ML:** Python, LibROSA, TensorFlow/PyTorch, Scikit-learn, CNNs, RNN/LSTM  
**Backend:** Python, Flask/FastAPI, REST APIs  
**Frontend:** HTML, CSS, JavaScript
